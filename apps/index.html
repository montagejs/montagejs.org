Tips & Tricks (Chrome OS)

Your first experience powering up a Chrome OS device is Tips & Tricks, a hosted MontageJS application with a custom loader. The application was designed and built in four and a half months.

The topics are presented using the MontageJS Flow component, which places arbitrary components along a custom 3D BÃ©zier path. Selecting a topic reveals more information in a lightbox-style interface while providing linear navigation cues on the sides. The Flow component also uses a frustum culling algorithm, which ensures that components hidden from view are not rendered, thus improving the application's initial load and runtime performance.

Note: Tips & Tricks requires a browser with WebM support. For best experience view this app in Google Chrome.

Scratchpad (Chrome)

Scratchpad 3.0 was the first MontageJS application we shipped. It added new features to and updated the visual design of the original Scratchpad notebook Chrome extension developed for Google.

The modular structure of MontageJS components and HTML5 templates made it easy for a team of five to work in parallel to create the skeleton of the app in one day. In addition to reducing development time, using MontageJS also resulted in deployment code 25% smaller in size than the original code.
